###### ASTROLOGER CRM #################


Astrologer CRM is a web-based customer relationship management system designed for astrologers to efficiently manage clients, schedule
consultations, track consultation status, and monitor business activities through a centralized dashboard.
The application helps astrologers organize client information, maintain consultation records, and gain quick insights into their practice 
through dashboard analytics.


##### FEATURES #####################
### AUTHENTICATION: 
~ User Registration
~ Secure Login
~ JWT-based Authentication
~ Protected Routes

### CLIENT MANAGEMENT
~ Add New Clients
~ View Client Details
~ Search Clients
~ Edit Client Information
~ Delete Clients

### CONSULTATION MANAGEMENT
~ Schedule Consultations
~ View Consultation History
~ Update Consultation Status
~ Delete Consultations

### DASHBOARD ANALYTICS
~ Total Clients
~ Total Consultations
~ Completed Consultations
~ Upcoming Consultations
~ Recent Clients Overview


###### TECH STACK #################
### FRONTEND  
React, CSS, Axios, React router DOM

### BACKEND
Node.js, Express,js,MongoDb, Mongoose , JWT Authentication, Bcrypt.js
